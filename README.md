# Online Marketplace
A Java client-server marketplace application supporting 
concurrent buyers and sellers.

## Features
- Multi-user accounts for buyers and sellers
- Store and product management
- Shopping cart and purchase history
- Persistent data storage across sessions

## Built With
Java · Swing · Socket Programming · Multithreading · JUnit

## Run
1. Run `Server.java`
2. Run `MarketDriver.java` (multiple instances supported)
